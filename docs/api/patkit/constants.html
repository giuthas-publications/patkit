patkit.constants

This module contains all sorts of constants used by patkit.

Enums are used for constants that need to be instantiated from other variables. They maybe used as fields in other objects. Using an Enum limits the possible values and avoids typos and makes an IDE help in writing code.

Frozen dataclasses are used for constants that only ever need to be accessed and never are stored. In effect, they function as look-up tables.

302class SplineMetaColumn(Enum):
303    """
304    Basic metadata that any Spline should reasonably have.
305
306    Accepted values:
307    - ignore: marks a column to be ignored, unlike the others below, 
308        can be used several times
309    - id: used to identify the speaker, 
310        often contained in a csv field called 'family name'
311    - given names: appended to 'id' if not marked 'ignore'
312    - date and time: dat3 and time of recording
313    - prompt: prompt of recording, used to identify the recording with 'id'
314    - annotation label: optional field containing annotation information
315    - time in recording: timestamp of the frame this spline belongs to
316    - number of spline points: number of sample points in the spline used 
317        to parse the coordinates and possible confidence information    
318    """
319    IGNORE = "ignore"
320    ID = "id"
321    GIVEN_NAMES = "given names"
322    DATE_AND_TIME = "date and time"
323    PROMPT = "prompt"
324    ANNOTATION_LABEL = "annotation label"
325    TIME_IN_RECORDING = "time in recording"
326    NUMBER_OF_SPLINE_POINTS = "number of spline points"

347class SplineShapesEnum(ListablePrintableEnum, metaclass=ValueComparedEnumMeta):
348    """
349    Spline metrics that characterise shape.
350    """
351    CURVATURE = 'curvature'
352    FOURIER = 'fourier'
353    MODIFIED_CURVATURE = 'modified_curvature'
354    PROCRUSTES = 'procrustes'
355
356    def short_name(self) -> str:
357        """
358        Return the short name or abbreviation of this SplineShape metric.
359
360        Returns
361        -------
362        str
363            The short name.
364        """
365        match self:
366            case SplineShapesEnum.CURVATURE:
367                return "CI"
368            case SplineShapesEnum.FOURIER:
369                return "Fourier"
370            case SplineShapesEnum.MODIFIED_CURVATURE:
371                return "MCI"
372            case SplineShapesEnum.PROCRUSTES:
373                return "PROC"
