patkit.constants

This module contains all sorts of constants used by patkit.

Enums are used for constants that need to be instantiated from other variables. They maybe used as fields in other objects. Using an Enum limits the possible values and avoids typos and makes an IDE help in writing code.

Frozen dataclasses are used for constants that only ever need to be accessed and never are stored. In effect, they function as look-up tables.

171class PatkitConfigFile(str, Enum):
172    """
173    Human written yaml files to control importing data.
174
175    Please note, that while MAIN corresponds to `patkit.yaml` which is the
176    conventional name for the file containing paths/names of other config files
177    such as data, gui, publish, and simulation config, and those files have
178    conventional names (`patkit_data.yaml` etc.), these filenames are only a
179    convention. The MAIN here is more of a guess of what we should look for
180    than a hard rule, and the rest should be specified in `patkit.yaml` instead
181    of PATKIT trying to guess their names.
182
183    DATA: specifications for processing data and deriving new Modalities and
184        Statistics.
185    GUI: specifications for gui elements - which graphs to display, color
186        scheme etc.
187    PUBLISH: specifications for publishing graphs
188    SIMULATION: specifications for simulating data and running analysis on the
189        simulated data
190    MANIFEST: list of Scenarios relating to a set of recorded data saved with
191        recorded data
192    SESSION: how PATKIT should read a session based on recorded data
193    SPLINE: spline formatting
194    """
195    EXERCISE = "patkit-exercise.yaml"
196    DATA = "patkit-data.yaml"
197    GUI = "patkit-gui.yaml"
198    PUBLISH = "patkit-publish.yaml"
199    SIMULATION = "patkit-simulation.yaml"
200    MANIFEST = "patkit-manifest.yaml"
201    SESSION = 'session-config.yaml'
202    SPLINE = 'spline-config.yaml'
203
204    def __str__(self) -> str:
205        """
206        Ensure f-strings and print() output the string value.
207        """
208        return self.value

327class SplineMetaColumn(Enum):
328    """
329    Basic metadata that any Spline should reasonably have.
330
331    Accepted values:
332    - ignore: marks a column to be ignored, unlike the others below,
333        can be used several times
334    - id: used to identify the speaker,
335        often contained in a csv field called 'family name'
336    - given names: appended to 'id' if not marked 'ignore'
337    - date and time: dat3 and time of recording
338    - prompt: prompt of recording, used to identify the recording with 'id'
339    - annotation label: optional field containing annotation information
340    - time in recording: timestamp of the frame this spline belongs to
341    - number of spline points: number of sample points in the spline used
342        to parse the coordinates and possible confidence information
343    """
344    IGNORE = "ignore"
345    ID = "id"
346    GIVEN_NAMES = "given names"
347    DATE_AND_TIME = "date and time"
348    PROMPT = "prompt"
349    ANNOTATION_LABEL = "annotation label"
350    TIME_IN_RECORDING = "time in recording"
351    NUMBER_OF_SPLINE_POINTS = "number of spline points"

372class SplineShapesEnum(ListablePrintableEnum, metaclass=ValueComparedEnumMeta):
373    """
374    Spline metrics that characterise shape.
375    """
376    CURVATURE = 'curvature'
377    FOURIER = 'fourier'
378    MODIFIED_CURVATURE = 'modified_curvature'
379    PROCRUSTES = 'procrustes'
380
381    def short_name(self) -> str:
382        """
383        Return the short name or abbreviation of this SplineShape metric.
384
385        Returns
386        -------
387        str
388            The short name.
389        """
390        match self:
391            case SplineShapesEnum.CURVATURE:
392                return "CI"
393            case SplineShapesEnum.FOURIER:
394                return "Fourier"
395            case SplineShapesEnum.MODIFIED_CURVATURE:
396                return "MCI"
397            case SplineShapesEnum.PROCRUSTES:
398                return "PROC"
